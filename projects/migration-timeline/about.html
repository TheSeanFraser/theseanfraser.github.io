About Project

This project is intended to predict when the highest number of each species starts to arrive in each region of Ontario. The methodology based on Mike Burrell's blog post. Instead of using eBird data from every available year, only the past 20 years (2002 - 2022) were used to have a more accurate prediction based on current circumstances.

All bird data is from eBird.

eBird. 2021. eBird: An online database of bird distribution and abundance [web application]. eBird, Cornell Lab of Ornithology, Ithaca, New York. Available: http://www.ebird.org. (Accessed: April 5 & June 9 2022).
